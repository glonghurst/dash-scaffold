# dash-scaffold

Static front-end scaffold for yougotdiss.com an external gameplay-clip ingest alpha (DASH).

Current state: static UI shell only (v0.2). No auth, no uploads, no backend everything in index.html is inert placeholder markup with inline comments marking future integration points (auth, upload handling, pipeline telemetry, metadata binding, return package, donation module).

Deployed via GitHub Pages to yougotdiss.com (custom domain via GoDaddy DNS).
